<Info>
  **前提条件** 您应该已安装Node.js（版本18.10.0或更高）。
</Info>

步骤1. 在您的操作系统上安装Mintlify：

<CodeGroup>

```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

步骤2. 进入文档所在位置（您可以找到`mint.json`的地方）并运行以下命令：

```bash
mintlify dev
```

文档网站现在可以在`http://localhost:3000`访问。

### 自定义端口

Mintlify默认使用3000端口。您可以使用`--port`标志自定义Mintlify运行的端口。例如，使用此命令在3333端口运行：

```bash
mintlify dev --port 3333
```

如果您尝试在已被占用的端口上运行Mintlify，将看到类似这样的错误：

```md
Error: listen EADDRINUSE: address already in use :::3000
```

## Mintlify版本

每个CLI都链接到特定版本的Mintlify。如果您的本地网站看起来与生产环境不同，请更新CLI。

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## 故障排除

以下是使用CLI时解决一些常见问题的方法。

<AccordionGroup>
  <Accordion title="Mintlify未加载">
    更新到Node v18。运行`mintlify install`并重试。
  </Accordion>
  <Accordion title="Windows上没有此类文件或目录">
前往`C:/Users/Username/.mintlify/`目录并删除`mint`文件夹。然后在此位置打开Git Bash并运行`git clone https://github.com/mintlify/mint.git`。

重复步骤3。

  </Accordion>
  <Accordion title="遇到未知错误">
    尝试导航到设备根目录并删除~/.mintlify文件夹。然后再次运行`mintlify dev`。
  </Accordion>
</AccordionGroup>
