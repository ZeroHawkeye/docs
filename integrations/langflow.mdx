## 在 Langflow 工作流中同步网络数据

Firecrawl 可以在 [Langflow，AI 工作流构建器](https://www.langflow.org/) 内部使用。本页介绍如何在 Langflow 内配置和使用 Firecrawl 模块。

<img
  className="block"
  src="/images/fc_langflow_block.png"
  alt="Firecrawl Langflow 模块"
/>

### 使用 Firecrawl 模块进行网页抓取

1. 登录您的 Firecrawl 账户并获取 API 密钥，然后在模块上输入或从工作流的其他部分传入。
2. (可选) 连接 Text Splitter。
3. 选择抓取模式以获取单个页面。
4. 输入要抓取的目标 URL 或从工作流的其他部分传入。
5. 根据您尝试获取的网站和数据设置任何页面选项和提取器选项。您也可以从工作流的其他部分传入这些选项。
6. 在您的工作流中使用数据。
