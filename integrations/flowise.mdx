## 在 Flowise 工作流中同步网络数据

Firecrawl 可以在 [Flowise 聊天流构建器](https://flowiseai.com/) 内部使用。本页介绍如何在 Flowise 内配置和使用 Firecrawl 模块。

<img
  className="block"
  src="/images/fc_flowise_block.png"
  alt="Firecrawl Flowise 模块"
/>

### 使用 Firecrawl 模块进行网站爬取

1. 登录您的 Firecrawl 账户并获取 API 密钥，然后在模块上输入。
2. (可选) 连接 Text Splitter。
3. 选择爬取模式以获取目标 URL 下的爬取页面。
4. 输入要爬取的目标 URL。
5. 在您的工作流中使用生成的文档。

### 使用 Firecrawl 模块进行网页抓取

1. 登录您的 Firecrawl 账户并获取 API 密钥，然后在模块上输入。
2. (可选) 连接 Text Splitter。
3. 选择抓取模式以获取单个页面。
4. 输入要抓取的目标 URL。
5. 在您的工作流中使用生成的文档。
